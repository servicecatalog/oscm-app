# What is it?
This repository introduces Maven to build and manage OSCM Asynchronous Provisioning Proxy related modules.

# How to use it?
Simply run 
`mvn clean install`
to get following artifacts:
- oscm-app.ear
- oscm-app-openstack.ear
- oscm-app-aws.ear
- oscm-app-azure.ear
- oscm-app-vmware.ear
